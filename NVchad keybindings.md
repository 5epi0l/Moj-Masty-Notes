


### 1. To open a terminal

```bash
SPC + h or SPC + v
```



### 2. To split a window

```bash
vsp(vertical split) || sp(horizontal split)
```



### 3. To display the filetree

```bash
ctrl + n
```



### 4. To list themes

```bash
SPC + t + h
```



### 5. To create a file

```bash
a
```



### 6. To mark a file

```bash
m
```




### 7. To delete a file

```bash
d
```




### 8. To find a file

```bash
SPC + f + f
```



### 9.  To switch between buffers

```bash
tab (to cycle forward)
shift + tab (to cycle backward)
```
